# TS Helper README

A lightweight VS Code extension that adds extra functionalities.

## Features

### JSX Smart Select

- Selects the closest JSX element when the cursor is inside JSX
- Falls back to VS Code’s built-in Select to Bracket behavior when not inside JSX.

Default Shortcut: `Shift + Alt + Enter` 

#### Supported Files

- `.tsx`

- `.jsx`

In other file types, the extension defaults to VS Code’s bracket selection.

**Enjoy!**
